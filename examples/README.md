# Examples
You can find different examples in this folder on how to use the library. For ease of use they are ordered from most basic to more advanced difficulty. This document only states a brief description. The in-depth explanations are found in the source files as code comments.

**001_readme**
- Best for beginners
- Explains the minimal example from the main README file

**005_ienumerable**
- Introduces main type of library `IEnumerable<T>`
- Shows alternatives to lambda expressions
- Explains deferred execution
